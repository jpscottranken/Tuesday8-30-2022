                    ohnk Wael Chorfan • a year ago

                    The code below accomplishes what you want to do. In the example below, tenant_id in the users table is a foreign key to the id primary key in the tenants table.

                    knex.schema
                    .createTable('tenants', function (table) {
                    table.increments('id');
                    table.string('value', 255).notNullable();
                    })
                    .createTable('users', function (table) {
                    table.increments('id');
                    table.string('full_name', 255).notNullable();
                    table.integer('tenant_id').unsigned().notNullable();
                    table.foreign('tenant_id').references('id').inTable('tenants');
                    })
